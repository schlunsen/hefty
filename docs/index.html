Features

Why hefty?

A lightning-fast disk space analyzer that lives in your terminal. No GUI needed.

⚡

Blazingly Fast

Built in Rust with efficient recursive directory traversal. Scans thousands of files in seconds.

▣

Interactive Treemap

Visualize disk usage with a squarified treemap. See at a glance which files are eating your space.

☰

List Mode

Non-interactive output perfect for scripting, piping, and quick checks. Integrates into your workflow.

⚙

Configurable

Filter by minimum file size, limit results to top N files, and scan any directory you choose.

🎨

Beautiful TUI

Powered by Ratatui with colored output, vim-style navigation, and a responsive split-pane layout.

📦

Easy to Install

Available via Homebrew and Cargo. Works on macOS and Linux, Intel and ARM.

Installation

Get started in seconds

Homebrew Recommended

$ brew tap schlunsen/tap
$ brew install hefty

Cargo

$ cargo install --git https://github.com/schlunsen/hefty

From Source

$ git clone https://github.com/schlunsen/hefty.git
$ cd hefty && cargo build --release
Usage

Simple and powerful

# Interactive TUI with treemap
$ hefty ~

# List top 10 files over 100MB
$ hefty ~/projects --list -n 10 --min-size 100MB
